# IDP Scanner (Android)

An Android application demonstrating an AI-ready document processing workflow.

## Current Features (Phase 1)
- Single-activity architecture with Fragment navigation
- PDF selection using Android Storage Access Framework
- URI-based document handling (no permanent storage)
- Simulated AI processing with loading state
- Result screen placeholder for confidence score and decision

## Planned Features
- OCR and table extraction
- Confidence scoring
- Automated approval / review decision
- Backend AI integration

## Tech Stack
- Kotlin
- Android Jetpack (Navigation, Fragments)
- Material 3
